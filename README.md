PASSWORD CHECKER
==================================================================================================================================================================================
📝 Problem Statement:

In today’s digital world, passwords play a vital role in securing user accounts. Weak passwords can cause security risks and unauthorized access. The objective of this project is to create a Python program that checks whether a password is strong or weak based on specific security conditions.

🎯 Objectives:

To validate password strength using Python
To practice string handling techniques
To apply decision-making statements
To understand basic cybersecurity validation concepts

🛠 Tools and Technologies Used:

Programming Language: Python
Software: Python IDLE / VS Code
Operating System: Windows

📖 Project Description:

This program checks whether the entered password satisfies the following conditions:
Minimum 8 characters
At least one uppercase letter
At least one lowercase letter
At least one digit
At least one special character

If all conditions are satisfied, the password is considered Strong.

Otherwise, it is marked as Weak.
